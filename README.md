# React Tabs

Универсальный компонент табов, реализованный на **React** без сторонних библиотек.

Проект создан как самостоятельный UI-компонент и может использоваться в любых React-проектах.

## Возможности

- React + чистый JavaScript
- Без дополнительных зависимостей
- Поддержка нескольких наборов табов на странице
- Гибкая настройка через пропсы

## Параметры компонента

| Проп                | Описание                                   |
|--------------------|--------------------------------------------|
| `tabs`             | Массив табов `{ id, title, content }`      |
| `activeTab`        | ID активного таба                           |
| `onTabClick`       | Функция для изменения активного таба       |
